Cloud Based Tutor Chatbot

I designed and deployed the Estudiemás English-tutoring chatbot, built on GPT-4o mini and hosted as an Azure Web App with sign-in to prevent unauthorized access. I migrated the project from an initial Microsoft Teams plan to the web app due to packaging issues, curated lesson content (and FAQs) in Excel to train the assistant, and tuned settings to keep responses accurate while controlling token usage and throughput. The goal was to ease teachers’ after-hours load and give students fast, lesson-aligned answers; the project ran from January 14 to February 21, 2025, and includes testing for topic adherence, memory use, and open-question handling.
